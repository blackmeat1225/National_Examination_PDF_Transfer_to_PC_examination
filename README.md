# National_Examination_PDF_Transfer_to_PC_examination
https://sites.google.com/view/nationalexaminationpdftransfer
1.將National_Exam_PDF_reader.html 下載
2.到國家考試的題庫搜尋頁
  https://wwwq.moex.gov.tw/exam/wFrmExamQandASearch.aspx
3.將你要的考卷打包下載後，解壓縮，答案跟題目放在同一個資料夾，請勿更動檔名。
4.開啟  National_Exam_PDF_reader.html
5.點選左上方選擇檔案，其實是選資料夾，開啟第3.資料夾
